# Day-of-the-Week-Champ
Game that tests your knowledge of a given dates day-of-the-week


## Setup:
```
python3 -m venv .venv
source .venv/bin/activate
pip3 install --upgrade pip
pip3 install -r requirements.txt
```

## Usage: 
```./game.py```
